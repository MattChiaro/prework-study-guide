HTML

  • The Head element contains information (Metadata) about the website
  • The Body element contains all the visible content
  • use the Title element in the head to optimize the page for SEO
  • you can use Anchors to link images to the page
  • Semantic elements are always preffered, when possible

CSS

  • all elements exist within a box. This is called the box model!
  • margin is how much space we want around the outside of an element
  • padding is how much space is around the content inside the element
  • use DRY: Don't Repeat Yourself!

Git

  • Always use command line!
  • git status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
