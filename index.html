HTML

  • The <head> element contains information (Metadata) about the website
  • The <body> element contains all the visible content
  • use the <title> element in the head to optimize the page for SEO
  • you can use anchors (<a>) to link images to the page
  • Semantic elements are always preferred, when possible

CSS

  • all elements exist within a box. This is called the box model!
  • margin is how much space we want around the outside of an element
  • padding is how much space is around the content inside the element
  • use DRY: Don't Repeat Yourself!

Git

  • Always use command line!
  • status: Checks what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git add -A: stages your changes
  • git commit -m "your message here": commits to the changes locally with a message describing them
  • git pull origin main: makes sure your local branch is up to date before pushing the changes. This pulls the main branch into your feature branch
  • git push origin branch-name: Pushes the changes from your local machine to the remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Functions are sets of instructions that tell the computer how to perform a certain task
